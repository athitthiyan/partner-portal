# StayEase Partner Portal

Standalone Angular partner dashboard for hotel owners on StayEase.

## Scripts

- `npm run start`
- `npm run build`
- `npm run lint`
- `npm run test`

## Core flows

- partner registration
- partner login
- hotel profile and GST/bank settings
- room inventory management
- booking inbox
- availability calendar
- revenue and payout summary
